# Come contribuire a RescueCom

## Hai trovato un bug?

- Apri una [issue](https://github.com/KronosPNG/RescueCom/issues)

## Come fare dei cambiamenti al codice

- I messaggi di commit dovranno seguire lo stile dei [conventional commit](https://www.conventionalcommits.org/en/v1.0.0/).

Tutorial rapido: [Stylesheet](https://gist.github.com/qoomon/5dfcdf8eec66a051ecd85625518cfd13)

1. Crea una nuova branch con un nome esplicativo

Esempio:

```bash
git checkout -b [nome-branch]
```

2. Aggiungi la modifica con `git add`

3. Scrivi un messaggio di commit

4. Pusha la branch

```bash
git push -u origin feat/new-feature
```

5. Apri una pull-request su GitHub
